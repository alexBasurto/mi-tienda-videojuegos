# Playzone: tienda de videojuegos
![PLAY ZONE](logo_bn.png)

## API Rest

Este repositorio incluye los archivos que componen el **backend** para la aplicación web PlayZone, una web para gestionar nuestra tienda de videojuegos. Accede al repositorio del frontend [aquí](https://github.com/ibaivalente/AngularVideojuegos).

Esta aplicación ha sido desarrollada usando tecnología ASP\.NET Core 8, como parte del proyecto final del Certificado de Profesionalidad de "Desarrollo de Aplicaciones web con Angular & .NET".

## Tecnologías

- Microsoft SQL Server
- Microsoft ASP\.NET Core 8.0
